# Cleanup Context

Reduce token usage and focus context on current task.

## Instructions

1. Summarize the current conversation state in 3-5 bullet points
2. Identify the active task/goal
3. List only the files currently being worked on
4. Clear any redundant historical context
5. Provide a focused summary for continuation

## Output Format

### Current State
- [summary points]

### Active Task
[current goal]

### Relevant Files
- [list of files]

### Ready to Continue
[confirmation]
